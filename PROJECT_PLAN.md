
# AI工具网站 - 完整项目规划 (2026)

## 一、项目定位与目标

### 1.1 网站类型
**AI工具评测与导航站**

### 1.2 核心目标
- 3个月内被Google收录并获得基础流量
- 6个月内通过Google AdSense开始盈利
- 1年内建立稳定的内容生态

---

## 二、技术架构

### 2.1 技术栈
| 维度 | 推荐方案 | 理由 |
|------|---------|------|
| 框架 | Next.js 14 + App Router | SEO友好，SSR/SSG支持好 |
| 样式 | Tailwind CSS | 开发效率高，响应式方便 |
| 内容管理 | MDX + 文件系统 | 内容管理灵活 |
| 部署 | Vercel | 全球CDN，速度快 |
| 统计 | Google Analytics 4 + Search Console | 追踪流量和收录 |

### 2.2 站点结构
```
/                    首页
/tools              工具列表
/tools/[slug]      工具详情
/categories/[category]  分类页
/reviews            评测文章
/blog               博客
/about              关于
/contact            联系
```

---

## 三、SEO核心策略

### 3.1 技术SEO
- **INP优化** &lt; 200ms
- **LCP** &lt; 2.5s
- **CLS** &lt; 0.1
- 生成sitemap.xml和robots.txt
- 结构化数据（Schema.org）
- 允许AI爬虫（GPTBot、Google-Extended等）

### 3.2 内容策略
- **E-E-A-T强化：作者档案、专业资质展示
- **AI+人工**：AI生成占比≤30%，必须人工审核和案例佐证
- **页面长度**：工具详情≥800字，评测文章≥1500字
- **更新频率**：每周2-3篇高质量内容
- **结构化内容**：FAQPage、ComparisonTable

### 3.3 关键词策略
- **核心词**：AI工具、AI写作工具、AI绘图工具
- **长尾词**：免费AI工具推荐、AI工具对比、AI工具测评
- **商业词**：AI工具会员优惠、AI工具折扣

---

## 四、Google AdSense 要点

### 4.1 申请条件
- 至少30-50篇高质量原创文章
- 完整的About/Contact页面
- 无违规内容
- 稳定的内容更新历史

### 4.2 广告位规划
- 侧边栏广告
- 文章末尾广告
- 工具卡片之间广告
- 相关文章区域

### 4.3 合规要点
- 不点击自己的广告
- 不诱导点击
- 遵守Google AdSense政策

## 六、风险与应对

| 风险 | 应对策略 |
|------|---------|
| 内容同质化 | 强化本地化、真实体验、对比表格 |
| Google算法更新 | 坚持白帽SEO，不关键词堆砌 |
| 收录慢 | 保证内容质量，主动提交索引 |
| 广告收益周期长 | 前3-6个月坚持持续运营 |
| AI内容降权 | AI+人工模式，标注人工审核 |

## 七、盈利模式

1. **Google AdSense** - 展示广告
2. **Affiliate联盟营销** - 部分AI工具提供返佣
3. **付费收录/置顶** - 后期
